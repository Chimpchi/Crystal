About

Crystal is a lightweight and bloat-free code editor for C/C++ designed to have familiar yet elegantly implemented features from other code editors.

Features - [x] Typing - [x] File Explorer - [x] Syntax highlighting (for C++) - [x] Lightweight - [x] Code suggestions - [x] Build error markers (compile-time) - [x] Basic terminal - [ ] Drag and drop files - [ ] Image viewer - [ ] Customizable themes
